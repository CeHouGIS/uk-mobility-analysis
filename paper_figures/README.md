# Paper figures (Fig 1–4)

Faithful reproduction of the figures in the source notebooks `ML.ipynb` and
`triangle_plot.ipynb`, generated by `run_all_figures.py`
(run from a directory containing the data files + a writable `fig/`).

| Group | Figures | Data source | Notes |
|-------|---------|-------------|-------|
| **Fig 1** binscatter | `Fig 1a–1f *_binscatter.png` | `Englnad_ML_merge_binscatter.csv` (wide) | per amenity type t1/t3/t5/t6/t8/t9; quantile binscatter Feb-2020 vs Mar-2020/2021 |
| **Fig 2** coefficient | `Fig 2a–2d coefficient_plot.png` | `graph_data/Figure_2*_data.csv` | coefficient ± 95% CI error bars |
| **Fig 3** ternary | `Fig 3a–3c ternary_distribution_*.jpg` | `England_ML 2.csv` (long) | workplace / amenity / home frequency shares, per period |
| **Fig 4** clustering | `Fig 4a–4c Cluster_heatmap_*.png`, `Fig 4d–4e heatmap_nature.jpg` | `England_ML.csv` (long) | KMeans k=5 on z-scored frequencies; per-period cluster mean heatmaps + transition matrices |

Notes
- The notebooks reassign `df` between sections: the binscatter uses the **wide**
  merge table, while clustering/Fig 4 needs the **long** table with a `month` column.
- Fig 1 panels t3/t6/t8/t9 are commented out in the source notebook; reproduced here
  for completeness (all six panels exist in the reference `fig/`).
- KMeans (`random_state=42`) finds the same five clusters as the reference; the cluster
  *index numbering* can be permuted across sklearn/BLAS versions (values match to rounding).
- Data files are not included in this repository.
